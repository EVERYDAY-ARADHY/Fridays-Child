# Project: Poem & Art Storing Gallery (Cutecore)

## Vision
A personal gallery for poems and art that feels like a cross between a vintage Tamagotchi and a soft 3D Y2K aesthetic. 

## Design Requirements
- **Theme**: Coquette Tamagotchi x Soft 3D Y2K.
- **Style Keywords**: Cutecore, 8-bit, Baby Pink, Pixelated, Neomorphism, Claymorphism, Glass UI.
- **Target**: Responsive (Mobile & Desktop).

## Screen List
1. **Landing/Home**: Hero section with a Tamagotchi-style frame, featured poem/art, and navigation.
2. **Gallery Feed**: Grid of poems and art pieces using claymorphic cards with pixelated borders.
3. **Single Poem View**: An immersive reading experience using glass UI overlays and soft 3D icons.
4. **Art Detail View**: Large image display with metadata and a "pixel art" style comment section.
5. **Upload/Creation Studio**: A playful interface for adding new poems or art, styled like a handheld game menu.
